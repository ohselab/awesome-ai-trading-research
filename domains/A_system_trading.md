## 1. Lo (2004) — 적응적 시장 가설 (Adaptive Markets Hypothesis)

| 항목 | 내용 |
|------|------|
| **전체 제목** | The Adaptive Markets Hypothesis: Market Efficiency from an Evolutionary Perspective |
| **저자** | Andrew W. Lo |
| **연도** | 2004 |
| **게재지** | Journal of Portfolio Management, 30(5), 30th Anniversary Issue, pp. 15-29 |
| **DOI** | `10.3905/jpm.2004.442611` |
| **SSRN ID** | 602222 |
| **인용수** | ~1,416회 (Semantic Scholar 기준) |

### 초록 요약
효율적 시장 가설(EMH)과 행동재무학 간의 오래된 갈등을 해소하기 위해 **적응적 시장 가설(Adaptive Markets Hypothesis, AMH)**을 제안한다. 진화 생물학의 원칙 — 경쟁, 돌연변이, 번식, 자연선택 — 을 금융시장에 적용한다. 시장 효율성은 정적 상태가 아닌 **동적·상황의존적** 결과물이다. 개인은 완전한 합리적 최적화자가 아니라 Herbert Simon의 "만족추구자(satisficer)"로서, 경험을 통해 학습하고 진화한다. 행동재무학이 "비합리적"이라고 분류한 편향들은 다른 진화 환경에서 형성된 적응적 휴리스틱으로 재해석될 수 있다. 시장 참여자는 서로 다른 "종"으로 모델링되며, 시장 효율성은 이들의 수, 경쟁 구도, 환경에 따라 결정된다.

### 핵심 기여
- **EMH와 행동재무학의 진화적 통합**: 두 패러다임의 상충을 진화론적 프레임워크로 해소. 시장 효율성은 "있거나 없는 것"이 아니라 **시간에 따라 변하는 스펙트럼**
- **시장 생태계 모형**: 시장 참여자를 "종"(연기금, 헤지펀드, 개인투자자 등)으로, 수익 기회를 희소 자원으로 모델링. 다양성이 시장 안정성의 원천
- **전략의 순환적 특성**: 투자 전략 수익성은 환경 의존적이며 주기적으로 변동. 혁신과 적응이 생존의 핵심

### 시스템 트레이딩에의 의미
시스템 트레이딩 전략이 **정적으로 고정되면 시장 적응으로 수익성이 소멸**함을 이론적으로 정당화한다. 적응형 알고리즘, 앙상블 전략 로테이션, 온라인 학습 기반 시스템의 학술적 근거를 제공한다. "어제의 알파는 오늘의 노이즈"라는 실무 직관에 진화론적 토대를 부여한다.

## 2. Lo, Mamaysky & Wang (2000) — 기술적 분석의 기초

| 항목 | 내용 |
|------|------|
| **전체 제목** | Foundations of Technical Analysis: Computational Algorithms, Statistical Inference, and Empirical Implementation |
| **저자** | Andrew W. Lo, Harry Mamaysky, Jiang Wang |
| **연도** | 2000 |
| **게재지** | Journal of Finance, 55(4), pp. 1705-1765 |
| **DOI** | `10.1111/0022-1082.00265` |
| **NBER WP** | w7613 |
| **인용수** | ~792회 (AFA 웹사이트 기준) |

### 초록 요약
기술적 분석(차트 패턴 분석)을 정량적 금융의 영역으로 끌어올리기 위해 **비모수 커널 회귀(kernel regression)** 기반 패턴 인식 알고리즘을 개발한다. 1962–1996년 NYSE/AMEX 350개 종목과 나스닥 종목의 일간 수익률을 대상으로, 헤드앤숄더·더블탑/바텀·삼각형·직사각형 등 **10가지 기하학적 패턴**을 알고리즘으로 정의하고 자동 탐지한다. 패턴 완성 후 3일간의 조건부 수익률 분포를 비조건부 분포와 비교하여, 카이제곱 검정·KS 검정·부트스트랩(1,000회)으로 통계적 유의성을 검증한다.

### 핵심 기여
- **기술적 분석의 알고리즘화**: 주관적 차트 해석을 커널 회귀 기반 자동 탐지 시스템으로 전환. 10가지 패턴을 수학적으로 정의하여 재현 가능하게 구현
- **통계적 검증 프레임워크**: NYSE/AMEX 7/10 패턴, 나스닥 **10/10 패턴**이 수익률 분포를 통계적으로 유의하게 변화시킴을 입증. 거래량 상승 패턴에서 정보 가치가 더 강함을 발견
- **시장별 차이 발견**: 나스닥에서 패턴의 정보적 가치가 더 강하며, 이는 시장 구조 차이(딜러 vs. 전자거래)와 연관

### 시스템 트레이딩에의 의미
기술적 분석의 **학술적 정당성**을 부여한 논문. 머신러닝·딥러닝 기반 패턴 인식, 자동화 차트 분석 시스템의 **이론적·방법론적 기초**를 제공한다. 커널 회귀 bandwidth 선택 문제(과적합 vs. 과소적합)는 현대 ML 하이퍼파라미터 튜닝과 동일한 구조적 난제를 보여준다.

## 3. Almgren & Chriss (2001) — 포트폴리오 거래의 최적 실행

| 항목 | 내용 |
|------|------|
| **전체 제목** | Optimal Execution of Portfolio Transactions |
| **저자** | Robert Almgren, Neil Chriss |
| **연도** | 2001 |
| **게재지** | Journal of Risk, 3(2), pp. 5-39 |
| **DOI** | `10.21314/JOR.2001.041` |
| **인용수** | ~2,402회 (Google Scholar 기준) |

### 초록 요약
포트폴리오 거래 실행 시 **변동성 위험**과 **시장 충격**(영구적·일시적 마켓 임팩트)으로 인한 거래 비용의 조합을 최소화하는 문제를 다룬다. 선형 비용 모형(영구 충격: g(v)=γv, 일시 충격: h(v)=ε·sgn(v)+(η/τ)·v) 하에서 시간 의존적 청산 전략의 **효율적 프론티어**를 명시적으로 구축한다. 거래의 자연 시간 스케일인 **"반감기(half-life)" θ=1/κ**를 도입하고, 실적 발표 등 예정된 뉴스 이벤트에 대한 분할 최적 전략을 개발한다. 유동성 조정 VaR(L-VaR)을 제안하여 리스크 측정에 유동성 비용을 통합한다.

### 핵심 기여
- **거래 실행의 효율적 프론티어**: 기대 비용 E[x]와 분산 V[x] 간의 최적 교환 비율을 보여주는 볼록한 프론티어를 해석적으로 구축. 위험회피 계수 λ에 따른 연속적 전략 계열 도출
- **거래 반감기(Half-life)**: θ=1/κ로 정의되는 고유 시간 척도. T≫θ이면 초기 집중 거래, T≪θ이면 균등 분할에 수렴. "나이브 전략"(등속 분할)은 위험회피 거래자에게 결코 최적이 아님을 증명
- **L-VaR (Liquidity-adjusted Value at Risk)**: 유동성 비용을 통합한 VaR. 프론티어의 접점에서 최적 전략 선택 가능

### 시스템 트레이딩에의 의미
**알고리즘 주문 실행 분야의 바이블**. VWAP, TWAP, IS(Implementation Shortfall) 등 실무 실행 알고리즘의 학술적 기반을 제공한다. 대량 주문의 시장 충격을 최소화하는 **최적 분할 전략** 설계의 이론적 틀이며, 모든 기관용 알고리즘 트레이딩 시스템의 핵심 참고 자료이다.

## 4. O'Hara (1995) — 시장 미시구조 이론

| 항목 | 내용 |
|------|------|
| **전체 제목** | Market Microstructure Theory |
| **저자** | Maureen O'Hara (Cornell University, Johnson Graduate School of Management) |
| **연도** | 1995 (초판, Blackwell) / 1998 (Wiley 재출판) |
| **출판사** | Blackwell Publishers / John Wiley & Sons |
| **쪽수** | 304쪽 |
| **ISBN** | `0631207619` / `9780631207610` |
| **인용수** | ~3,622회 (Google Scholar 기준) |

### 내용 요약 (단행본 — 초록 대신 핵심 내용 기술)
시장 미시구조 분야의 가장 포괄적인 이론서로, 가격 형성 메커니즘을 체계적으로 분석한다. **7개 장**으로 구성: (1) 재고 모형 (2) 정보 기반 모형 (3) 정보를 가진 거래자 (4) 정보와 가격 과정 (5) 시장 생존 가능성과 안정성 (6) 유동성과 시장 간 관계 (7) 시장 성과 이슈. Kyle(1985), Glosten-Milgrom(1985) 등의 선행 모형을 통합하고 확장하여, 정보 비대칭 하에서의 가격 발견, 호가 스프레드 형성, 유동성 공급 메커니즘을 포괄적으로 설명한다. Bayesian 학습과 합리적 기대 프레임워크를 광범위하게 활용한다.

### 핵심 기여
- **정보 비대칭 모형의 체계적 통합**: Kyle 모형(전략적 거래자), Glosten-Milgrom 모형(순차적 거래), 재고 기반 모형(Stoll, Ho-Stoll 등)을 하나의 이론적 프레임워크로 정리·확장
- **호가 스프레드의 결정 요인 분해**: 정보 비용, 재고 비용, 주문 처리 비용이 스프레드에 미치는 영향을 분리 분석하고, 각 모형의 함의를 체계적으로 비교
- **시장 구조와 유동성의 관계**: 딜러 시장, 경매 시장, 전자 거래소 등 다양한 시장 구조에서의 가격 발견과 유동성 차이를 이론적으로 설명

### 시스템 트레이딩에의 의미
시스템 트레이딩 전략이 **작동하는 시장의 내부 메커니즘**을 이해하는 데 필수적인 이론서. 호가 스프레드, 유동성, 정보 흐름, 주문서 dynamics를 이해해야 효과적인 전략 설계가 가능하다. 고빈도 트레이딩(HFT), 마켓메이킹 알고리즘, 스마트 주문 라우팅의 이론적 기반을 제공한다.
