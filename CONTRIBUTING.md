# Contributing to Awesome AI Trading Research

Thank you for your interest in contributing!

## How to Contribute

### Suggest a Paper
1. Open an issue with the paper title, arXiv ID or DOI
2. Include a brief description of why it's noteworthy
3. Specify which domain (A1-C5) it belongs to

### Submit a Pull Request
1. Fork the repository
2. Add the paper to the appropriate domain file in `domains/`
3. Follow the existing format (table row with: Paper, Authors, Year, Venue, Citations, Link)
4. If Wow Score ≥ 80, also add to `wow-papers/README.md`
5. Submit PR with description

### Wow Score Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Practical Applicability | 30% | Can it be applied to real trading strategies? |
| Novelty | 25% | How does it differ from existing approaches? |
| Performance | 25% | Improvement over benchmarks |
| Reproducibility | 15% | Code/data availability |
| Timeliness | 5% | Relevance to current market conditions |

## Code of Conduct

- Be respectful and constructive
- Cite original sources properly
- Do not share proprietary or copyrighted content
- Focus on academic merit and practical value
